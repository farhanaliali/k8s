How to create EKS cluster With eksctl 


Install eksctl

		curl --silent --location "https://github.com/weaveworks/eksctl/releases/latest/download/eksctl_$(uname -s)_amd64.tar.gz" | tar xz -C /tmp
		sudo mv /tmp/eksctl /usr/local/bin
		eksctl version

Configuration and credential file settings
	
For example, the files generated by the AWS CLI for a default profile configured with aws configure looks similar to the following.

~/.aws/credentials

[default]
aws_access_key_id=AKIAIOSFODNN7EXAMPLE
aws_secret_access_key=wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY

~/.aws/config

[default]
region=us-west-2
output=json



or use 
		aws configure


		

Now create cluster 
		
		eksctl create cluster --name rvt-staging --version 1.21 --region ap-south-1 --nodegroup-name linux-node --node-type t2.micro --nodes 1
